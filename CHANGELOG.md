## [v0.1.0] - 2025-12-31

### Highlights

- Initial release of Pulumi components for deploying agentkit-based agents to AWS Bedrock AgentCore

### Added

- Native Go Pulumi components for AWS Bedrock AgentCore deployment ([`9edbbfe`](https://github.com/plexusone/agentkit-aws-pulumi/commit/9edbbfe))
- Fluent builder API (`AgentBuilder`, `StackBuilder`) matching agentkit-aws-cdk patterns ([`9edbbfe`](https://github.com/plexusone/agentkit-aws-pulumi/commit/9edbbfe))
- Configuration loading from JSON or YAML files via `NewStackFromFile` ([`9edbbfe`](https://github.com/plexusone/agentkit-aws-pulumi/commit/9edbbfe))
- VPC creation with public/private subnets across multiple AZs ([`9edbbfe`](https://github.com/plexusone/agentkit-aws-pulumi/commit/9edbbfe))
- NAT Gateway for private subnet internet access ([`9edbbfe`](https://github.com/plexusone/agentkit-aws-pulumi/commit/9edbbfe))
